# MemTest — Universal Evaluation Toolkit for AI Memory Systems

> System-agnostic evaluation library. Provide a memory system, get an evaluation report.

## Design Principles

- **Zero Dependencies**: Pure Python standard library + JSON. No coupling with NOESIS or any specific system.
- **Adapter Pattern**: The system under test only needs to implement 3 methods (`store` / `search` / `reset`) to plug into the evaluation pipeline.
- **Standardized Data**: Evaluation data and evaluation logic are fully decoupled.

## Quick Start

```python
from memtest import MemoryTestSuite, MemoryAdapter, load_test_db

# 1. Define your adapter: tell the framework how your memory system works
class MyMemoryAdapter(MemoryAdapter):
    def reset(self):
        """Clear the database"""
        self.db.clear()

    def store(self, memory_text: str, metadata: dict):
        """Store a memory. metadata includes memory_id/time/location/person/event"""
        self.db.insert(text=memory_text, **metadata)

    def search(self, query: str, top_k: int = 20) -> list[dict]:
        """Retrieve memories, return [{memory_id, score}, ...]"""
        return self.db.query(query, limit=top_k)

# 2. Load evaluation data
db = load_test_db("sample_db_100.json")  # or generate your own

# 3. Run evaluation
adapter = MyMemoryAdapter()
suite = MemoryTestSuite(adapter)
report = suite.run(db)
print(report.summary())
```

## Evaluation Data

| Data | Source | Scale |
|------|--------|-------|
| `sample_db_100.json` | Procedurally synthesized, 100 entries | Ready to use |
| `test_db_10000.json` | Generated by `generator.py` | Full 6 categories |
| `your_books.json` | `knowledge_builder.py <corpus_dir>` | Custom |

## Output

The evaluation report covers 6 dimensions: storage integrity / retrieval precision & recall / organization clustering / forgetting rationality / logical reasoning / deep retrieval.
