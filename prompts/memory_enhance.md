# 提示词：记忆文本增强（3种表达变体）

## 任务
给定一条结构化记忆，生成同一事件的3种不同表达风格。

## 输入格式
```json
{
  "person": "张伟",
  "identity": "项目经理",
  "partner": "王芳",
  "partner_identity": "设计师",
  "relation": "同事",
  "location": "北京 星巴克 CBD核心区",
  "event": "购买 茅台 100股 单价1800",
  "time": "2026-01-15 14:30:00"
}
```

## 输出格式
```json
{
  "versions": [
    {"style": "标准叙述", "content": "..."},
    {"style": "详细描述", "content": "..."},
    {"style": "口语化", "content": "..."}
  ]
}
```

## 风格定义
- **标准叙述**：客观、简洁、新闻体，包含关键人物、时间、地点、事件
- **详细描述**：补充背景、心理活动、场景细节，像小说片段
- **口语化**：日常对话口吻，像朋友闲聊，用词随意自然

## 规则
1. 三种风格必须描述同一事件，核心事实完全一致
2. 不要编造输入中未提供的信息
3. 中文语境，自然流畅
4. 每种版本 30-120 字
